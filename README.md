# ScoreForm (得点入力 & 集計システム)

## 概要
イベント運営における班ごとの得点入力・自動集計、および Slack 上でのランキング発表をシームレスに行うためのシステムです。Google Apps Script (GAS) をバックエンドとし、Google スプレッドシートをデータベースとして活用します。

## 開発の背景
イベントなどの多人数・同時進行型イベントでは、各会場のスコアをリアルタイムに集計し、即座に結果をフィードバックすることが求められます。
手動集計によるミスやタイムラグを最小限に抑え、スタッフが「進行」に集中できるよう、**「入力のしやすさ」「二重送信の防止」「進捗の可視化」**に重点を置いて開発されました。

## 主な機能

### 1. スマートな入力フォーム (`index.html`)
- **ステップ式入力**: 班名・ふりがな・人数を確定（スプシへ先行保存）してから得点入力へ進む設計。
- **リアルタイムチェック**: 「ふりがな」欄にひらがな以外が入力されると即座に警告を表示。
- **連打対策**: 通信中はボタンを無効化（Disabled）し、物理的な二重送信を防止。

### 2. 管理者用モニタリング画面 (`?page=admin`)
- **進捗の可視化**: 全班の「入力完了/未完了」を一覧表示。
- **クイック集計**: 現在の合計点と完了班数をリアルタイムで確認可能。

### 3. Slack 連携 (Slash Commands)
- **`/得点`**: 実行したユーザーにのみ、フォームの URL をエフェメラルメッセージ（本人限定）で通知。
- **`/発表`**: 締切済みのセクションのランキング（上位3位）をチャンネル全体へパブリッシュ。
- **実行ロック**: `CacheService` を利用し、コマンドの連打（3秒以内）による API エラーを防止。

## 技術スタック
- **Language**: JavaScript (Vanilla JS)
- **Platform**: Google Apps Script (GAS)
- **Database**: Google Sheets
- **Communication**: Slack API (Slash Commands)
- **Styling**: CSS3 (Modern Responsive Design)

## セットアップ：スプレッドシートの構造

### 1. `index` シート
- **B1セル**: 現在操作対象のシート名（例：`1部`）
- **A2以降のセル**: 各セクション名
  - 各セッション名を入力してください。
- **B2以降のセル**: 各セクションのステータス
  - `締切` になると、Slack でのランキング発表やフォームでの順位閲覧が解禁されます。

### 2. `問題` シート（マスターデータ）
2行目以降に設定を入力します。A列の記号によってフォームの挙動が変化します。

| 列 | 項目 | 説明 |
| :--- | :--- | :--- |
| **A列** | **区分フラグ** | `*` : **大問（セクション）**。以降の小問をこの大問にグループ化します。<br>`-` : **数値入力モード**。フォームが自由入力形式になります。 |
| **B列** | **問題名** | 画面に表示される設問名（例：問1、ボーナスポイントなど）。 |
| **C列** | **得点 / Max** | 通常：正解時の加点。<br>数値入力時（`-`）：**入力可能な最大値**。 |
| **D列** | **失点 / Min** | 通常：不正解時の減点。<br>数値入力時（`-`）：**入力可能な最小値**。 |

## 2. GAS のデプロイ
1. GAS エディタに `コード.gs`, `index.html`, `admin.html` を作成し貼り付けます。
2. `コード.gs` 内の `webAppUrl` 変数に、デプロイ後に発行される URL を記述します。
3. 「デプロイ」>「新しいデプロイ」から、種類：ウェブアプリ、アクセスできるユーザー：全員 で公開します。

## 3. Slack API の設定
1. Slack App を作成し、Slash Commands を追加します。
2. `Request URL` に GAS のウェブアプリ URL を設定します。
3. 「Escape channels, users, and links sent to your app」は **OFF** に設定してください。

## 使い方
| URLパラメータ | 画面の種類 | 主な用途 |
| :--- | :--- | :--- |
| (なし) | **入力フォーム** | 現場スタッフによるスコア入力。 |
| `?page=admin` | **管理画面** | 本部による入力進捗のモニタリング。 |
| `?page=ranking` | **全順位画面** | 締切後の全班ランキング表示。 |
